# ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОГРАММНОГО ОБЕСПЕЧЕНИЯ

## Наименование программного продукта

Информационная система управления салоном красоты (ногтевой сервис).

## Назначение программного продукта

Программное обеспечение предназначено для автоматизации деятельности студий ногтевого сервиса и салонов красоты. Система обеспечивает онлайн-запись клиентов, управление расписанием мастеров, учет материалов, расчет начислений сотрудников, формирование отчетности и анализ основных показателей деятельности салона.

## Требования к программному обеспечению

Для запуска системы необходимо наличие следующего программного обеспечения:

- Docker Desktop;
- Docker Compose;
- Git;
- Visual Studio Code (рекомендуется);
современный веб-браузер (Google Chrome, Microsoft Edge, Mozilla Firefox).

## Получение проекта

Исходный код приложения размещается в локальном каталоге проекта. Перед запуском необходимо открыть каталог проекта в Visual Studio Code или другом редакторе исходного кода.

## Первый запуск системы

После открытия проекта необходимо выполнить следующие команды в терминале:

cp .env.example .env

docker compose up -d --build

docker compose exec app composer install

docker compose exec app php artisan key:generate

docker compose exec app php artisan migrate --seed

Указанные команды выполняют создание контейнеров Docker, установку зависимостей Laravel, генерацию ключа приложения, создание структуры базы данных и загрузку демонстрационных данных.

## Запуск клиентской части

Для сборки интерфейса приложения необходимо выполнить:

npm install

npm run dev

При использовании Laravel Sail запуск осуществляется командой:

./vendor/bin/sail npm run dev

## Доступ к системе

После успешного запуска приложение становится доступным по адресу:

http://localhost:8000

Вход в систему выполняется через страницу авторизации.

Для демонстрации работы используются учетные записи различных ролей пользователей:

администратор;
мастер;
клиент.

Логины и пароли задаются в процессе заполнения демонстрационных данных базы данных.

## Основные функции системы
После запуска пользователю доступны следующие возможности:

- онлайн-запись клиентов;
- управление расписанием и сменами мастеров;
- ведение клиентской базы;
- управление услугами;
- учет материалов и контроль остатков;
- автоматическое списание материалов;
- расчет начислений мастеров;
- работа с программой лояльности и промокодами;
- формирование аналитики и отчетов;
- экспорт данных в форматы Excel и PDF;
- просмотр уведомлений и статистики.

## Структура проекта

Проект реализован на основе фреймворка Laravel и включает следующие основные каталоги:

app — бизнес-логика приложения;
database — миграции и демонстрационные данные;
resources — шаблоны интерфейса и клиентские ресурсы;
routes — маршруты приложения;
public — публичные файлы веб-приложения;
storage — служебные данные и журналы работы системы.

## Остановка системы

Для завершения работы контейнеров Docker необходимо выполнить команду:

docker compose down

После остановки все данные сохраняются в базе данных и будут доступны при следующем запуске системы.
